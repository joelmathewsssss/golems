#Golems


The golems of Ankh Morpork are rising up under Adora after years of unfair labor. The City Watch must stop them by guessing a 4-element control spell. Each element is one of: Fire, Water, Air, Earth. Duplicates are allowed. After each guess, golem eyes provide clues: RED = correct element in the correct location. BLUE = correct element in the wrong location. Example: Secret FFAW, Guess FEWA -> RBB. Guess FAFW -> RRBB. You only get 10 guesses to disable the golems.
